Google Search Console Kurulumu
Adım 1 — Mülk Ekle

search.google.com/search-console → Başlayın
URL öneki seç → https://hotpulse.me yaz
DNS doğrulama yöntemini seç (en güvenilir)
Adım 2 — DNS Doğrulama (Namecheap / Cloudflare vb.)
Google sana şöyle bir TXT kaydı verecek:

google-site-verification=XXXXXXXXXXXXXXXXXXXX
DNS yöneticine git → TXT kaydı ekle:

Alan	Değer
Host	@ (veya hotpulse.me)
Value	google-site-verification=XXXX...
TTL	300
Kaydettikten sonra Google'a dön → Doğrula → 1–24 saat sürebilir.

Adım 3 — Sitemap Gönder
Sol menü → Dizine Ekleme → Site Haritaları → URL kutusuna yaz:

sitemap.xml
→ Gönder

Google artık tüm videolarını, kategorileri ve özel sayfaları otomatik keşfeder.

3️⃣ Bing Webmaster Tools (bonus — %30 ekstra trafik)
bing.com/webmasters → Başlayın
Google Search Console'u zaten doğruladıysan → GSC'den içe aktar seç → tek tıkla biter
Sol menü → Site Haritaları → https://hotpulse.me/sitemap.xml ekle
4️⃣ Mevcut SEO Durumun (Ne Var / Ne Eksik)
Özellik	Durum	Açıklama
/sitemap.xml	✅ Aktif	Videolar + kategoriler + özel sayfalar
/robots.txt	✅ Aktif	Botlara sitemap yolunu bildiriyor
Video sayfası Open Graph	✅ Aktif	og:title, og:image, og:description
Twitter Card	✅ Aktif	summary_large_image formatı
Canonical URL	✅ Aktif	Video sayfasında doğru ayarlı
Dinamik <title>	✅ Aktif	Video adı + "— Soci" formatında
JSON-LD yapısal veri	❌ Eksik	Google'da zengin sonuçlar için gerekli
Server-side meta injection	⚠️ Kısmi	JS ile yapılıyor; bot gecikmesine dikkat
5️⃣ ⚠️ Önemli: React SPA + SEO Sorunu
Siteniz React tabanlı (SPA). Meta taglar JavaScript ile enjekte ediliyor. Google bunu okuyabilir ama yavaş (1–2 hafta gecikmeli). Facebook/WhatsApp/Twitter paylaşım önizlemeleri hiç görmez çünkü bu botlar JavaScript çalıştırmaz.

Çözüm olarak şunu yapabilirim (bir komutla):

Django'ya video URL'lerini (/videos/<slug>) bot isteği olarak algılayıp sunucu tarafında og: meta taglarını HTML'e gömecek bir middleware ekleyebilirim. Yani:

Googlebot → /videos/my-video-slug
Django → Video'yu DB'den çeker → og:title, og:image vb. <meta> etiketlerini HTML'e gömer → Gönderir
Normal kullanıcı → Normal React SPA → JS ile meta ayarlanır
Bu sayede hem Google hem de sosyal medya paylaşımları doğru önizleme gösterir.

7️⃣ Kontrol Listesi (VDS'e Geçince)
□ backend/.env → SITE_URL=https://hotpulse.me ayarlı
□ DNS → TXT kaydı ile Google doğrulandı  
□ Search Console → sitemap.xml gönderildi
□ Bing Webmaster → sitemap.xml gönderildi
□ robots.txt erişilebilir → hotpulse.me/robots.txt
□ sitemap.xml erişilebilir → hotpulse.me/sitemap.xml
□ İlk indexleme → 3–7 gün içinde başlar
